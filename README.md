# Istio Service Mesh 
creating istio service mesh for K8S cluster, for fine grain traffic routing and security for service to service communicaiton and authenication & autherization of services in microservice architecture

# Gateway 
For exposing application over internet gateway is used

# Virtual Service
For routing traffic to services inside the cluster using routing rules like defining service and type of service 

# Destination Rules
for fine grade traffic routing, policies for routing traffic inside the cluster

# mTLS (security)
using peer authenication for service to service communication with authentication and authorization for security

# Authorization policies 
fine grained access control between services 

# Observability (kiali, Prometheus, Grafana) 
Traffic Graph through Kiali
